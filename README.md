# Data Science Learning Repository

A structured learning repository for SQL, Python, and various Data Science tools and processes. Clone this repo in [Positron](https://positron.posit.co/) to get started immediately.

## Directory Structure

```
data-science/
├── sql/                  # SQL learning materials
│   ├── resources/        # Copied tutorials, references, and examples
│   └── projects/         # Your own SQL projects and exercises
├── python/               # Python learning materials
│   ├── resources/        # Copied tutorials, references, and examples
│   └── projects/         # Your own Python scripts and projects
├── data-science-tools/   # Data Science tools & workflows (pandas, scikit-learn, etc.)
│   ├── resources/        # Copied tutorials, references, and examples
│   └── projects/         # Your own data science projects and analyses
├── notebooks/            # Jupyter (.ipynb) and Quarto (.qmd) notebooks
└── shared-resources/     # Files and references copied from other repositories
```

## Getting Started in Positron

1. **Clone this repo** and open the folder in [Positron](https://positron.posit.co/).
2. **Set up your Python environment** — see `requirements.txt` for a starter list of packages. Run:
   ```bash
   pip install -r requirements.txt
   ```
3. **Explore each folder** — every directory has a `README.md` with suggestions on what to add and recommended resources.
4. **Start a notebook** — open or create a `.ipynb` or `.qmd` file in the `notebooks/` folder.
