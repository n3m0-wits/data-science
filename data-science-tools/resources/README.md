# Data Science Tools — Resources

Place here any files, scripts, or materials you copy from other repositories or sources related to data science tools and workflows.

## What to Put Here

- Cheat sheets for ML libraries (`scikit-learn`, `xgboost`, etc.)
- Example notebooks from courses or tutorials
- Pre-built pipeline templates
- Model evaluation code snippets

|----------|-----------|------|

## Organization Tips

- Prefix resources by topic: `sklearn-pipeline-template.py`, `eda-starter.ipynb`
- Keep a `notes.md` inside topic sub-folders summarizing key takeaways from resources you've reviewed
