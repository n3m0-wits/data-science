# SQL Resources

Place here any files, scripts, or materials you copy from other repositories or sources.

## What to Put Here

- Cheat sheets (e.g., `sql-cheatsheet.md`, `window-functions-reference.md`)
- Example scripts from tutorials or courses
- Starter schemas and seed data files (`.sql` files)
- Reference implementations you want to keep handy

## Suggested Resources to Copy / Download

| Resource | What It Is | Link |
|----------|-----------|------|
| `joins-cheatsheet.md` | Visual guide to all JOIN types | Create your own from [Visual JOIN](https://joins.spathon.com/) |
| `window-functions.sql` | Example queries for all window functions | From [Mode's window function tutorial](https://mode.com/sql-tutorial/sql-window-functions/) |
| `duckdb-quick-start.sql` | DuckDB local analytics examples | [DuckDB docs](https://duckdb.org/docs/sql/introduction) |

## Organization Tips

- Keep files named descriptively: `topic-subtopic.sql` (e.g., `joins-left-outer.sql`)
- Add a comment header to every `.sql` file explaining what it demonstrates
- Group larger collections into sub-folders (e.g., `resources/postgresql/`, `resources/duckdb/`)
