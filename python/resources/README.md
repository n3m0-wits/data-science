# Python Resources

Place here any files, scripts, or materials you copy from other repositories or sources.

## What to Put Here

- Cheat sheets (e.g., `pandas-cheatsheet.md`, `python-basics-reference.py`)
- Example scripts from tutorials or courses
- Utility modules you collect and want to reuse
- Config templates (e.g., `logging.conf`, `pyproject.toml` starter)

## Suggested Resources to Copy / Download

| Resource | What It Is | Link |
|----------|-----------|------|
| Pandas cheat sheet | Quick reference for DataFrame operations | [Official PDF](https://pandas.pydata.org/Pandas_Cheat_Sheet.pdf) |
| NumPy cheat sheet | Array operations at a glance | [DataCamp PDF](https://www.datacamp.com/cheat-sheet/numpy-cheat-sheet-data-analysis-in-python) |
| Matplotlib gallery | Example plot snippets | [matplotlib.org/gallery](https://matplotlib.org/stable/gallery/) |

| Python regex reference | Common regex patterns for data cleaning | Create from [regex101.com](https://regex101.com/) |

## Useful GitHub Repos to Clone / Reference

- [jakevdp/PythonDataScienceHandbook](https://github.com/jakevdp/PythonDataScienceHandbook) — Notebooks for the free textbook
- [wesm/pydata-book](https://github.com/wesm/pydata-book) — "Python for Data Analysis" notebooks (Wes McKinney, pandas creator)
- [fastai/fastbook](https://github.com/fastai/fastbook) — Deep learning with fastai (advanced)

## Organization Tips

- Name helper scripts with a clear purpose: `fetch_data.py`, `clean_csv.py`
- Add a module docstring at the top of every `.py` file
- Group resources by topic: `resources/pandas/`, `resources/visualization/`
