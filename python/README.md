# Python

This section covers Python for data science, scripting, and automation. Use it for both reference material and hands-on practice.

## Subdirectories

| Folder | Purpose |
|--------|---------|
| [`resources/`](resources/README.md) | Tutorials, cheat sheets, and examples copied from other repos or sources |
| [`projects/`](projects/README.md) | Your own Python scripts, notebooks, and projects |

## Suggested Learning Path

1. **Python Basics** — Data types, control flow, functions, list/dict comprehensions
2. **File I/O & Standard Library** — Reading/writing files, `os`, `pathlib`, `csv`, `json`
3. **Data Manipulation** — `pandas` for DataFrames, `numpy` for arrays
4. **Data Visualization** — `matplotlib`, `seaborn`, `plotly`
5. **Working with APIs** — `requests`, JSON parsing, authentication
6. **Data Cleaning** — Handling missing values, type coercion, string operations
7. **Statistical Analysis** — `scipy.stats`, descriptive statistics, hypothesis testing
8. **Machine Learning Basics** — `scikit-learn` for regression, classification, clustering
9. **Environment Management** — `venv`, `pip`, `requirements.txt`, `pyproject.toml`

## Key Libraries to Know

| Library | Use Case |
|---------|----------|
| `pandas` | DataFrames, data cleaning, reshaping |
| `numpy` | Numerical arrays and math |
| `matplotlib` / `seaborn` | Static charts and statistical plots |
| `plotly` | Interactive charts |
| `scikit-learn` | Machine learning |
| `scipy` | Scientific computing and statistics |
| `requests` | HTTP / API calls |
| `sqlalchemy` | Database connections |
| `duckdb` | Fast in-process analytics (great with pandas) |
| `polars` | Fast DataFrame alternative to pandas |
